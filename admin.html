GitHub Personal Access Token

The admin uses your GitHub token to read and write posts directly to your repository. The token is stored in your browser's session storage and persists until you close the tab.

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click Generate new token (classic)
  3. Give it a name like "BBT Admin"
  4. Select scope: repo (full repo access)
  5. Click Generate and copy the token
